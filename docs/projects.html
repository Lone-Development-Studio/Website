  • Smart Home System

    Technology is at the point in which we could control every aspect of the house with our phones. But many smart home device manufactures have to accommodate so many different scenarios that smart home devices are not in every aspect of the home.
    Like seen in many futuristic movies this project serves to try and add a micro controller to every aspect of the home.

    Just so that all the devices I design are compatible and buildable, I will make and install all of the device within my own home.
    All the code and model files will be listed within the github repo. There will most likely be many revisions to all the devices as well developing a new device, I might add more features to an exiting device.

    This Project could range as far as fully functioning fridges that can tell you what is in it at any given time to a custom built wall that houses displays on both sides. The Complexity of the system might reach a point where the amount of data available to all the device so much that a I would probably need to build custom network routing system.

  • Planatary Survivor

    After crash landing on a alien planet your job as the player is to either survive or thrive.
    Survival is the basic principal of the game. While playing the game the play can become very well established that they could survive on the planet for longer than the need to, thus they are thriving on the planet.

    Crashland...Escape, seems like most survival games, well Planatary Survivor aims to cater to most choices the player could make.
    If the player wants to build in space they can, else if they want to build on the planet they can. Being as flexable as possible is what I will strive to achive with this game.
